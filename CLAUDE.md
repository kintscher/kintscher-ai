# Kintscher.ai — Landing Page

## Purpose
Single-route digital business card. Title + subtitle + socials. Nothing else.

## Design System
- **Background:** #0d0d14 (deep near-black)
- **Foreground:** #e5e7eb
- **Muted:** #6b7280
- **Accent:** #60a5fa
- **Mono font:** JetBrains Mono (use for title + link labels)
- **Sans font:** Inter (use for subtitle + body)
- **Aesthetic reference:** Linear, Vercel, Rauno Freiberg. Dense whitespace, precise typography, subtle depth. No gradients on text. No "glow" effects. No emoji.

## Constraints
- Single `index.astro` route.
- No client-side JavaScript. Zero. If a detail requires JS, drop the detail.
- No external images. No external icons — if icons are needed, inline SVG only.
- Must score 100/100/100/100 on Lighthouse.
- Must work at 320px width and 4K without media-query proliferation.

## Tone
Confident, understated. No marketing language. No "passionate about" anything.
